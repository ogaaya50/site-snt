Projet SNT : Internet et le Web

Dans ce site, nous allons découvrir les bases du Web et apprendre comment créer des pages internet grâce aux langages HTML et CSS. Le Web est un ensemble de pages accessibles via Internet.

L’objectif de ce projet est de comprendre comment fonctionne un site web et comment structurer et styliser une page.

Ce que nous allons voir :

  • Les bases du HTML
  • Le rôle du CSS
  • Une introduction au JavaScript
