Les bases du développement Web

Un site web repose sur trois langages principaux : HTML, CSS et JavaScript.

HTML

HTML (HyperText Markup Language) permet de structurer une page web. Il sert à organiser le contenu avec des titres, des paragraphes, des images, des liens, etc.

CSS

CSS (Cascading Style Sheets) permet de styliser la page web : couleurs, tailles, disposition, design.

JavaScript

JavaScript permet d’ajouter de l’interactivité à un site web, comme des animations ou des boutons dynamiques.

Étapes de création d’un site :

  1. Créer la structure avec HTML
  2. Ajouter le style avec CSS
  3. Ajouter des interactions avec JavaScript
